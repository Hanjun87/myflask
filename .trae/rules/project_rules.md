# AI 行为规则

## 禁止事项

### 严禁使用 Emoji
- **禁止在代码中使用任何 emoji 字符**
- **禁止在注释中使用 emoji**
- **禁止在文档中使用 emoji**
- **禁止在字符串/输出内容中使用 emoji**

### 违规示例（禁止）
```python
# 错误示例
user_avatar = '👨‍💻'  # 禁止：使用emoji作为头像
print("✅ 操作成功")     # 禁止：使用emoji作为图标
# 🎉 庆祝一下         # 禁止：注释中使用emoji
```

### 正确示例
```python
# 正确示例
user_avatar = ''  # 不使用任何代替emoji
# 完成：功能已实现       # 纯文字注释
```

## 代码风格要求

1. **所有文本内容必须使用纯文字描述**
2. **界面图标使用 Font Awesome 或 SVG 代替 emoji**
3. **用户头像使用文字标识符或图片 URL**
4. **状态指示使用文字标签或 CSS 样式**

## 检查清单

在提交代码前，请确认：
- [ ] 代码文件中不包含任何 emoji 字符
- [ ] 注释中不包含任何 emoji 字符
- [ ] 字符串常量中不包含任何 emoji 字符
- [ ] 配置文件（如用户数据）中不包含 emoji 作为标识符
